# Frontend - Veredicta AI Dashboard

MVP inicial do dashboard (Módulo 1: Overview) em React + TypeScript + Vite.

## Rodar local

1. Copie `.env.example` para `.env` (se necessário).
2. Instale dependências:
   - `npm install`
3. Inicie o servidor:
   - `npm run dev`

Frontend padrão: `http://127.0.0.1:5173`
API padrão: `http://127.0.0.1:8011`
